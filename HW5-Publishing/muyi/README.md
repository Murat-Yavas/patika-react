# muyi

> Made with create-react-library

## Install

```bash
npm install --save muyi
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'muyi'
import 'muyi/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
